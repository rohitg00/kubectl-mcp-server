What You Can Do

Talk to your clusters like you talk to a DevOps expert

🔍

Debug & Troubleshoot

Analyze crashed pods, inspect logs, diagnose network issues, and get AI-powered root cause analysis with actionable fixes.

💰

Optimize Costs

Detect over-provisioned resources, get rightsizing recommendations, and identify idle workloads wasting cloud spend.

🛡

Audit Security

RBAC permission auditing, security policy scanning (Kyverno/Gatekeeper), certificate management, and vulnerability assessment.

⚙

Manage Helm Charts

Full Helm v3 lifecycle: install, upgrade, rollback, search repos, inspect values, and manage releases across clusters.

🌐

Multi-Cluster Operations

Switch contexts, compare deployments across clusters, manage kind local clusters, and vCluster virtual clusters.

📊

Interactive Dashboards

8 interactive UI tools: pods, logs, deployments, Helm, cluster overview, cost analysis, events, and network topology.
